# CWS/POS/NER

Chinese word segmentation, Part-of-speech tagging and Medical named entity recognition From scratch.

## Task Description

### Directory Structure

* Data: (each has its _cws, _pos, _ner file)
  * devset
  * testset1
  * trainset
* Evaluation
  * pos_evaluate.py
  * ner_evaluate.py
